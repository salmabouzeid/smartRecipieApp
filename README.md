Our app:
  Smart Recipe App
Description:
  The Smart Recipe App is a web application that helps users discover recipes based on the ingredients they already have at home.
  Users can enter available ingredients and the system will suggest recipes, cooking instructions, and possible ingredient substitutions using AI.
  The goal is to make cooking easier and help reduce food waste.
Features:
    •	Enter available ingredients
    •	AI-generated recipe suggestions
    •	Ingredient substitution recommendations 
    •	Recipe instructions 
    •	Basic nutrition information
    •	Recipe history saved in the database
Tech stack:
  •	Frontend: HTML, CSS, JavaScript
  •	Backend: Python + Flask
  •	AI: Google Gemini API
  •	Database: MongoDB Atlas
  •	DevOps: GitHub, GitHub Actions, Docker
  •	Deployment: Render
